## О боте

Бот работает в Telegram: в общем чате команды и в личных сообщениях. Пишите обычным русским языком. Он опирается на Google-таблицу, календарь и документы на Drive.

**Главное:**

- Поручения и напоминания сотрудникам — в **личку** с ботом.
- Общий чат — вопросы команде, объявления, эскалации.
- «Запомни» — долгие правила; бот видит недавние реплики из других чатов.

## Подготовка (один раз)

### Сотрудники

1. В таблице бота, лист **employees**: имя как в разговоре («Иван»).
2. Каждый: личка с ботом → **`/start`**.
3. Без `/start` личные поручения не дойдут.

### Рабочий чат

1. Бот в группе → **`/chatid`** → скопировать число в лист **chats**, **active=true**.

### График

Синхронизация из вашей таблицы (вкладка **«График Текущий месяц»**) каждый день в **07:00** (МСК), если бот запущен. Спрашивайте: «Кто сегодня на смене?», «Кто завтра работает?»

### База знаний (меню, регламенты, инструкции)

1. Сложите материалы в **папку Google Drive** (настраивает разработчик; бот имеет доступ на чтение).
2. Удобные форматы: **Google Документ**, текст, **PDF**.
3. Каждый день в **08:00** (МСК) бот сам обновляет индекс; после загрузки нового файла можно написать: **«Обнови базу знаний»**.
4. Спрашивайте обычным языком: «Что в регламенте про …?», «Какие вегетарианские блюда в меню?» — бот ищет по файлам в папке, **не выдумывает**.
5. Отдельный документ по ссылке (редко): «Прочитай документ id …» — если нужен один файл вне общей папки.

## Команды

| Команда | Где | Зачем |
|---------|-----|--------|
| `/start` | Личка | Привязка сотрудника |
| `/chatid` | Группа | ID чата для таблицы |

Остальное — обычным текстом.

## Поручения

**Руководитель:** формулирует поручение → бот пишет сотруднику в личку с ID задачи.

**Сотрудник:** отвечает **реплаем** на сообщение бота (текст или фото). При чеклисте — фото с разбором пунктов. Несколько задач — кнопки выбора.

**Закрытие:** руководитель «Одобри задачу …» или «Верни на доработку …».

---

## Быстрый тест (5 пунктов)

1. Поручение в личку сотруднику.
2. Сотрудник — реплай «сделал» или фото.
3. Две задачи — «готово» без реплая (кнопки).
4. Сообщение в общий чат → вопрос в личке про этот чат.
5. «Напиши в общий чат: …»
6. «Обнови базу знаний» → вопрос по тексту из папки Drive.

---

## Google Tasks и календарь

**Google Tasks (колонки Ира, Августа…):** поручение из Telegram дублируется в список сотрудника в Google Tasks + сообщение в личку. Имена в таблице `employees` должны совпадать с названиями списков.

**Календарь:**
- новые мероприятия бот пишет в календарь **«Мероприятия»**;
- «что сегодня» — из **основного** календаря и **«Мероприятия»** вместе.

## Автоматически по расписанию

| Время (МСК) | Что делает бот |
|-------------|----------------|
| **07:00** | Подтягивает график смен из вашей таблицы |
| **08:00** | Обновляет базу знаний из папки Drive |
| **Каждые 5 мин** | Напоминания по задачам и ваши автоматизации |

Бот должен быть **запущен** в это время (на сервере или на вашем ПК в режиме разработки).

## Если не работает

| Ситуация | Проверка |
|----------|----------|
| Нет ЛС сотруднику | `/start`, telegram_user_id в employees |
| Не то имя | Имя в таблице = как в фразе боту |
| Нет ответа по меню | Файлы в папке Drive; «Обнови базу знаний»; формат Google Doc или PDF |
| Нет графика | Заполнена вкладка «График Текущий месяц»; бот работал в 07:00; иначе попросите разработчика запустить синк |
| База не обновилась | Бот работал в 08:00 или напишите «Обнови базу знаний» |

При баге: скрин, время МСК, текст запроса, ID задачи из ЛС.
